Pipeline Task 
-------------

Store pipeline tasks. At Current step, you will know which tool is running. 

.. image:: ../img/workflow/pipeline_task_1.png
    :align: center

* Click |info-circle| icon to show info task.
* Click |times-circle| icon to cancel the task when the task is running.
* Click |play-circle| icon to continue task when fully loaded token (task will be paused when not enough token)
