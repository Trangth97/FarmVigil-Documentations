Shared with me
--------------

Where storing and managing shared vectors. You can search the folder, vector by name or order by date, name.

Click the tab Shared with me to start.

Make copy 
=========

Click |copy| icon on vector -> Click Confirm button. Copied vector named: name + “copy”

Delete 
======

Click trash icon -> Click Confirm button

    :align: center

===========

Click eye icon to view image in map
